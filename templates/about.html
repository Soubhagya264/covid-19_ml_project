What is the Idea of Creating this model ? Covid-19 Detector

The idea is to stop the transmission by prioritize tests and hence detecting the case quickly.Data can be collected on the symptoms of Covid-19. A Machine learning model is then trained on the data to find out the probability of a person having infection.

Machine Learning Model : How it's work?

This machine learning model used logistic regression algorithm to find out whom test for the infection first under a limited testing capacity.
